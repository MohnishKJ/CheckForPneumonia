# PneumoniaDoc 🩺

AI-powered web app for rapid pneumonia detection from chest X-ray images.

## 🚀 What it does
- Upload a chest X-ray image and get instant pneumonia classification
- Gives health recommendations based on the result
- Built for quick use by both healthcare professionals and patients

## ⚙️ How it works
- Uses a fine-tuned VGG19 model trained via transfer learning
- Applies data augmentation during training for better generalization
- Processes uploaded X-rays using OpenCV before prediction
- Flask backend serves the model and handles requests
- Simple web interface for image upload and instant results

## 🛠️ Tech Stack
- Python
- HTML, CSS, JavaScript
- Flask
- TensorFlow
- Keras
- OpenCV
- NumPy

## 💡 Use case
Built to give a fast, preliminary pneumonia check from X-ray images without waiting on manual review.
